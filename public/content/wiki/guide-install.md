# StarMC 保姆级教程：从零开始安装您的星际终端

如果您是第一次接触 Minecraft 或者不清楚如何安装我们的专属环境，本教程将手把手带您完成所有准备工作。

## 第一步：下载启动器 (Launcher)
我们推荐以下两款主流启动器：
- **PCL2 (推荐)**: 界面精美，功能最全，适合新手。
- **HMCL**: 稳定、简洁，适合老玩家。
- *下载地址均可在官方 Q 群文件找到。*

## 第二步：配置 Java 环境 (JDK)
这是游戏能否启动的核心。
1. 下载 **Java 17** (或更高版本)。
2. 安装完成后，在启动器的设置中，将“Java 路径”指向您刚安装的目录（通常是 `javaw.exe`）。

## 第三步：安装游戏版本
1. 在启动器中选择“下载游戏”。
2. 搜索并选择 **1.21.1** (或当前服务器运行的版本)。
3. **重要**: 如果您想画面更漂亮，请选择安装“Fabric”或“Forge”加载器，并勾选“Iris”或“Optifine”补丁。

## 第四步：添加服务器
1. 启动游戏后，点击“多人游戏”。
2. 点击“添加服务器”。
3. 服务器名称填：`StarMC`。
4. 服务器地址填：`mc.star-mc.top`。

## 第五步：进入并注册
1. 双击服务器进入。
2. 按 `T` 键打开聊天框。
3. 输入 `/register 您的密码 您的密码`。

**恭喜您！** 您已经成功降临在 StarMC 的主城。接下来，请查阅《生存手册》开启您的冒险。

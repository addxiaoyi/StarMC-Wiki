# PCL2 启动器优化指南：让您的 StarMC 之旅更流畅

很多玩家反映游戏卡顿，其实 80% 的原因在于启动器设置不当。本指南以 PCL2 为例，教您如何进行“神级配置”。

## 1. 内存分配（最关键）
- 点击“设置” -> “游戏设置”。
- 不要让启动器“自动分配”内存。
- **推荐值**: 如果您的电脑有 16G 内存，建议给游戏分配 4096MB (4G) 或 6000MB (6G)。分配太多反而会导致系统卡顿。

## 2. 渲染器选择
- 如果您的显卡较旧，请在设置中尝试切换渲染模式（如从 OpenGL 切换到兼容模式）。

## 3. 自动下载补丁
- PCL2 有一个强大的功能：自动补丁。
- 进入服务器前，如果提示缺少某些资源，点击“一键修复”即可，无需手动寻找模组。

## 4. 离线皮肤设置
- 在“账号管理”中，点击您的离线账号。
- 您可以上传自己的皮肤文件，或者直接输入一个正版玩家的 ID 来获取其皮肤。

## 5. 整合包推荐
- 我们在群文件提供了一套“StarMC 极简优化包”，内置了大幅提升 FPS 的模组。直接将该压缩包拖入 PCL2 即可安装。

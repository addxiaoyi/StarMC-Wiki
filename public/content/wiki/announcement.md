# 舵星归途 StarMC 核心公告与全方位生存指引

欢迎来到 StarMC。作为一名初次降临星系的旅者，本公告将为您揭示服务器的核心动态、技术架构以及未来发展蓝图。我们建议您定期查阅此页面，以获取第一手的官方资讯。

## 第一部分：当前运行状态与资源获取

### 1. 群文件与资源中心
服务器的各类补丁、模组包、光影以及优化后的客户端，均已同步至官方 Q 群文件。如果您格在连接过程中遇到资源缺失或版本不匹配的问题，请优先前往群文件进行检索。我们设立了专门的“下载专区”，涵盖了从 Java 环境到高性能光影的所有必备工具。

### 2. 官网与实时监控
- **官方网站（开发中）**: [https://star-web.top](https://star-web.top) —— 我们的官网正在紧锣密鼓地进行 UI 重构，未来将集成在线地图、排行榜以及皮肤上传功能。
- **状态监控页**: [https://status.s3.fan/s/starmc](https://status.s3.fan/s/starmc) —— 这里可以实时查看各个子服务器的 TPS（每秒刻数）、在线人数以及延迟情况。如果遇到大规模网络波动，该页面是判断问题的权威依据。

## 第二部分：版本架构与跨服逻辑

### 1. 多版本兼容性说明
StarMC 采用了高度优化的 ViaVersion 跨版本协议。虽然我们的核心服务端运行在 1.21.x 环境下，但我们支持从 1.16.x 到最新 1.21.x 的所有客户端连接。
**专家建议**: 为了获得最佳的方块渲染和最新的游戏特性，我们强烈推荐使用 1.21.x 系列版本进入。这能有效避免低版本进入高版本服时出现的模型渲染异常。

### 2. 身份验证与登录安全
如果您在进入服务器时提示“请输入密码”但从未注册过，请注意：
- **正版玩家**: 服务器支持 Mojang 官方验证。如果您是正版账号但提示需要密码，请联系管理团队进行账号绑定，之后即可享受无感免密登录。
- **离线玩家**: 请更换一个尚未被注册的角色名。注册指令为 `/register 密码 密码`。请务必使用强密码，并妥善保管，服务器不负责找回因弱密码导致的账号丢失。

## 第三部分：社区文化与长期愿景

### 1. 我们的坚持
StarMC 自成立以来，始终坚持“非商业化运营”的初衷。我们没有强制性的氪金礼包，也没有破坏平衡的特权道具。所有的资源、称号和成就，都需要通过您在游戏内的汗水与智慧来获取。

### 2. 未来路线图
- **短期目标**: 完善“诺娜大陆”RPG 任务系统，引入更具挑战性的副本玩法。
- **中期目标**: 建立跨服统一的经济市场（目前正在施工中，详见相关章节）。
- **长期目标**: 打造一个真正意义上的“星系社会”，让每一位玩家的建筑和历史都能在这片星海中留下永恒的印记。

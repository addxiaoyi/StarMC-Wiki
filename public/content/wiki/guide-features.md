# StarMC 特色玩法大搜罗：不只是生存那么简单

如果您觉得普通的砍树挖矿有些枯燥，StarMC 还为您准备了丰富的“副菜”。

## 1. 粘液科技 (Slimefun) —— 在生存服里玩工业
- **开启方式**: 无需下载模组，通过插件实现。
- **玩法**: 输入 `/sf guide` 获取一本书。您可以制造发电机、自动采矿机、甚至是能飞行的喷气背包。
- **建议**: 从制作“简易工作台”开始您的科技树。

## 2. 诺娜大陆 (RPG) —— 史诗般的剧情体验
- 前往主城的“时空祭坛”传送点。
- 在这里您可以接受 NPC 的委托，探索神秘的地牢，并获得具有独特属性的“史诗级武器”。

## 3. 全球贸易 (Global Shop)
- 输入 `/ah` 随时随地逛超市。
- 您可以把野外挖到的珍稀附魔书挂在上面，换取星币来购买领地。

## 4. 宠物系统 (Pets)
- 参与节日活动或完成特定成就，可以获得跟随您的专属宠物。
- 它们不仅可爱，有些还能为您提供额外的背包空间。

## 5. 每日任务 (Daily Quests)
- 输入 `/quests` 查看今日目标。完成任务可获得大量经验值和物资奖励。
